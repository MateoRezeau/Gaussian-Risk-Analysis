Gaussian Risk CalculatorA professional, lightweight Python toolkit for calculating parametric Value at Risk (VaR) and Expected Shortfall (ES) to facilitate rapid portfolio risk assessment.  OverviewThis tool provides a command-line interface to estimate potential portfolio losses under a Gaussian distribution assumption. It is designed for financial analysts and researchers needing a quick, reliable method for sensitivity analysis and risk screening.  Financial ContextAs a finance professional with experience in credit risk analysis and quantitative research, I developed this calculator to serve as a streamlined utility for portfolio risk modeling. It bridges the gap between raw market data and actionable risk metrics, providing a foundational approach to assessing tail risk. Inputs: The script will prompt for the mean return, volatility (σ), confidence level, and the time horizon in days.  Mathematical DisclaimerThis tool utilizes the Parametric (Gaussian) Method. Please note the following limitations:  Normality Assumption: The model assumes returns are normally distributed and may underestimate "fat-tail" risks or the magnitude of extreme market events.  Volatility Scaling: The script applies the square-root-of-time rule (σ_h = σ × √T), which assumes returns are independent and identically distributed (i.i.d.).
